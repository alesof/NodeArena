## Parameters and Dynamic Configuration
**Parameters** in ROS2 are key-value pairs associated with a node that allow dynamic configuration.  
They can control the behavior of the node without requiring a restart.  
Using parameters, nodes can expose internal settings such as access codes, thresholds, or operational modes.  
This challenge teaches how parameters can influence node behavior, and how to interact with them programmatically or via CLI.
